# Personal Website & Digital Garden

A modern, performant personal website built with Astro, featuring a digital garden of knowledge, changelog tracking, and various interactive components.

## 🚀 Features

- **Blog Posts & Articles**: Write and organize content with markdown
- **Digital Garden**: Collection of interconnected notes and knowledge
- **Changelog**: Track development progress and updates
- **Image Gallery**: Responsive masonry layout with view transitions
- **Data Analytics Section**: Share Jupyter notebooks and analytics projects
- **Sailboating Guide**: Structured educational content
- **Anki Card Integration**: Educational flashcards for learning

## 🔄 Automated Deployments

The site automatically deploys to GitHub Pages through GitHub Actions:

- On push to master branch
- Daily rebuilds for dynamic content
- Manual triggers available

## 🎨 Styling

The project uses a custom theme with CSS variables for consistent styling. Key features include:

- Dark mode optimized
- Responsive design
- Glass morphism effects
- Smooth transitions
- Custom card layouts
